Prerequisites

  • Python 3.11+
  • Node.js 18+
  • Docker & Docker Compose
  • Git

Quick Setup

# 1. Fork and clone the repository
git clone https://github.com/del4pp/opentrace.git
cd opentrace

# 2. Start development environment
docker-compose up -d

# 3. Install frontend dependencies
cd frontend && npm install

# 4. Start frontend development server
npm run dev

Development Workflow

  1. Create a branch: git checkout -b feature/your-feature-name
  2. Make changes: Write code and tests
  3. Test locally: Ensure everything works
  4. Commit changes: Write clear commit messages
  5. Push to branch: git push origin feature/your-feature-name
  6. Create Pull Request: Describe your changes
