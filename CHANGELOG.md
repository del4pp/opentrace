## [1.0.0] - 2025-10-01

### Added
- **Core Analytics Features:**
  - Privacy-first web tracking (no cookies, no fingerprinting)
  - Lightweight JavaScript tracker (< 2KB)
  - Real-time dashboard with basic metrics
  - ClickHouse integration for high-performance analytics
  - PostgreSQL for metadata storage
  - Docker-based deployment
  - REST API for data access
  - Multi-tenant architecture (resources, campaigns, events)
  - Basic user authentication
  - Live activity map with real-time updates

- **Supported Platforms:**
  - Web applications
  - Telegram bots
  - Discord bots
  - Basic mobile app support

- **Documentation:**
  - Setup and installation guides
  - API reference documentation
  - Architecture and technical documentation

### Security
- GDPR and privacy compliance
- Secure authentication mechanisms
- Data encryption at rest and in transit
- Regular security audits and updates

## Types of changes
- `Added` for new features

- `Fixed` for any bug fixes
- `Security` in case of vulnerabilities

## Versioning Policy

This project uses [Semantic Versioning](https://semver.org/):

- **MAJOR** version for incompatible API changes
- **MINOR** version for backwards-compatible functionality additions
- **PATCH** version for backwards-compatible bug fixes

## Release Frequency

- **Major releases:** Every 6-12 months
- **Minor releases:** Every 1-2 months
- **Patch releases:** As needed for critical fixes
